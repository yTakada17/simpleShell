#Abstract
I make simple shell run in the terminal.

#Usage
* Compile source code

  `gcc -o shell shell.c`
* Run shell

  `./shell`

* End shell
 * Ctrl+c
 * Type `exit`or`quit`

#Command
* `exit` or `quit`

  End shell.
* `jobs` and `bg`
* 
  The same command as Linux.

* `ls`, `emacs`, and the like you usually use. You can add options like `ls -a`.

#Note
You can use `&` to run the program as backgroud process.
